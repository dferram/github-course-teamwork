📋 Instrucciones

Sigue estos pasos para agregar tu página a la galería.

1

Hacer Fork del repositorio

Entra al repositorio original en GitHub y haz clic en el botón Fork (esquina superior derecha). Esto crea una copia en tu cuenta personal.

2

Clonar tu Fork

Abre tu terminal y clona tu fork (no el original):

git clone https://github.com/TU-USUARIO/Teamwork.git cd Teamwork
3

Crear una rama nueva

Crea una rama con tu nombre para trabajar sin afectar main:

git checkout -b nombre-apellido
💡 Ejemplo: git checkout -b juan-perez
4

Crear tu página HTML

Copia la plantilla y renómbrala con tu nombre:

cp _plantilla.html nombre-apellido.html

Abre tu archivo y personalízalo: cambia el título, escribe sobre ti, agrega tus redes sociales. ¡Sé creativo/a!

💡 Puedes usar el styles.css incluido o escribir tu propio CSS.
5

Agregar tu tarjeta a la galería

Abre index.html y busca la sección GALERÍA. Agrega una nueva tarjeta con tu información:

<a href="nombre-apellido.html" class="gallery-card"> <div class="card-avatar">🧑‍💻</div> <h3>Tu Nombre</h3> <p>Una frase sobre ti</p> </a>
⚠️ Este paso puede generar un merge conflict porque varios alumnos editan el mismo archivo. ¡Es intencional y parte del aprendizaje!
6

Hacer commit y push

Guarda tus cambios y súbelos a tu fork:

git add . git commit -m "Agregar página de nombre-apellido" git push origin nombre-apellido
7

Crear un Pull Request

Ve a tu fork en GitHub. Verás un banner para crear un Pull Request. Haz clic en "Compare & pull request", escribe un título descriptivo y créalo.

8

Resolver merge conflicts (si los hay)

Si GitHub indica conflictos, actualiza tu rama:

git checkout main git pull upstream main git checkout nombre-apellido git merge main

Abre los archivos con conflictos, resuélvelos, y luego:

git add . git commit -m "Resolver merge conflicts" git push origin nombre-apellido
9

¡Listo! 🎉

Una vez que tu Pull Request sea aprobado y mergeado, tu página aparecerá en la galería del equipo. ¡Felicidades, acabas de colaborar en un proyecto real!
