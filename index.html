INSTRUCCIONES

Sigue estos pasos para agregar tu pagina a la galeria. Lee el README.md para instrucciones detalladas.

1

CLONAR EL REPOSITORIO

Abre tu terminal y clona el repositorio del curso:

git clone <URL_DEL_REPOSITORIO> cd Teamwork
[TIP] Tu instructor te dara la URL del repositorio.
2

CREAR TU RAMA

Crea una rama con tu nombre (minusculas, con guiones). NUNCA trabajes en main:

git checkout -b nombre-apellido
[TIP] Ejemplo: git checkout -b juan-perez
3

COPIAR LA PLANTILLA HTML

Copia la plantilla y renombrala con tu nombre. NO edites el contenido del HTML:

Copy-Item pages\_plantilla.html pages\nombre-apellido.html
[!] NO escribas tu nombre, avatar ni skills en el HTML. Todo se carga automaticamente desde team.js. Solo necesitas que el archivo exista con el nombre correcto.
[TIP] En Mac/Linux usa: cp pages/_plantilla.html pages/nombre-apellido.html
4

CREAR TU AVATAR 8-BIT

Abre avatar-builder.html en tu navegador. Elige un preset de la galeria o dibuja tu propio avatar pixel por pixel. Cuando termines, haz clic en COPIAR para obtener el codigo.

[TIP] Guarda el codigo en un bloc de notas por si lo necesitas despues. El codigo se ve asi: 555555555555555511555511...
5

REGISTRAR TUS DATOS EN TEAM.JS

Abre js/team.js, busca el comentario "AGREGA TU LINEA AQUI ABAJO" y agrega tu objeto con TODOS tus datos:

{ name: "Tu Nombre", page: "pages/nombre-apellido.html", tagline: "Tu frase", avatar: "TU_CODIGO", github: "https://github.com/tu-usuario", about: "Sobre ti", skills: ["HTML", "CSS"], randomFact: "Dato curioso" },

Reglas para evitar merge conflicts:
- Edita SOLO tu linea, nunca toques las de otros.
- Agrega tu objeto al final, debajo del comentario guia.
- NO borres el comentario "AGREGA TU LINEA AQUI ABAJO".
- Verifica que tu objeto termine con }, (coma obligatoria).

[!] El campo page debe coincidir EXACTAMENTE con el nombre de tu archivo HTML del paso 3. Si no coincide, tu pagina aparecera en blanco.
6

VERIFICAR LOCALMENTE

Abre index.html con Live Server o un servidor local y verifica que tu tarjeta aparezca en la galeria y tu pagina cargue bien.

[TIP] Si algo no funciona, abre la consola (F12) y revisa los errores. El problema mas comun es una coma faltante en team.js.
7

COMMIT Y PUSH

Guarda tus cambios y subelos a GitHub:

git add . git commit -m "Agregar pagina de nombre-apellido" git push origin nombre-apellido
8

CREAR UN PULL REQUEST

Ve al repositorio en GitHub. Veras un banner amarillo para crear un Pull Request. Haz clic en "Compare & pull request", escribe un titulo descriptivo y crealo.

9

RESOLVER MERGE CONFLICTS (SI LOS HAY)

Si GitHub indica conflictos, actualiza tu rama:

git checkout main git pull origin main git checkout nombre-apellido git merge main

Abre team.js, busca las marcas <<<<<<<, ======= y >>>>>>>. Borra esas 3 lineas y conserva AMBOS bloques de codigo (tu linea y la de tu companero):

git add . git commit -m "Resolver merge conflicts" git push origin nombre-apellido
10

MISION COMPLETA

Una vez que tu PR sea aprobado y mergeado, tu pagina aparecera en la galeria del equipo automaticamente. Acabas de colaborar en un proyecto real con Git.
